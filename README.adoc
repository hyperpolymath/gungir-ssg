== Overview

gungir-ssg provides a unified interface to *28 static site generators* across *17 programming languages*, enabling polyglot SSG operations via the Model Context Protocol (MCP).

[cols="1,3"]
|===
|Hub |https://github.com/hyperpolymath/poly-ssg-mcp[poly-ssg-mcp]
|Type |Satellite project
|Runtime |Deno 2.1+
|Protocol |MCP (Model Context Protocol)
|Compliance |RSR Gold
|===

== Quick Start

=== Prerequisites

[source,bash]
----
# Install Deno (required)
curl -fsSL https://deno.land/install.sh | sh

# Install just (recommended)
cargo install just
# or: brew install just
----

=== Installation

[source,bash]
----
git clone https://github.com/hyperpolymath/gungir-ssg.git
cd gungir-ssg

# Verify setup
just health-check

# List available adapters
just list-adapters
----

=== Basic Usage

[source,javascript]
----
// Import an adapter
import * as zola from "./adapters/zola.js";

// Connect to the SSG
await zola.connect();

// Use a tool
const result = await zola.tools
  .find(t => t.name === "zola_build")
  .execute({ path: "./my-site", drafts: true });

console.log(result);

// Disconnect
await zola.disconnect();
----

== Architecture

----
gungir-ssg (satellite)
    │
    ├── adapters/          # 28 SSG adapters
    │   ├── zola.js
    │   ├── hakyll.js
    │   └── ...
    │
    ├── *.scm              # SCM metadata files
    │   ├── META.scm       # Architecture decisions
    │   ├── ECOSYSTEM.scm  # Project relationships
    │   ├── STATE.scm      # Progress tracking
    │   ├── PLAYBOOK.scm   # Operations runbook
    │   ├── AGENTIC.scm    # AI integration
    │   └── NEUROSYM.scm   # Symbolic patterns
    │
    └── poly-ssg-mcp (hub) # Upstream adapter source
----

== Adapter Interface

Each adapter exports:

[source,javascript]
----
export const name = "SSGName";
export const language = "Language";
export const description = "Description";

export async function connect(): Promise<boolean>;
export async function disconnect(): Promise<void>;
export function isConnected(): boolean;

export const tools = [
  {
    name: "ssg_operation",
    description: "Operation description",
    inputSchema: { /* JSON Schema */ },
    execute: async (args) => { /* implementation */ }
  }
];
----

== Security

* All commands use array arguments (no shell injection)
* Environment variables for credentials
* Deno permission model enforced
* Weekly CodeQL scanning
* Dependabot enabled

Report vulnerabilities via https://github.com/hyperpolymath/gungir-ssg/security/advisories/new[GitHub Security Advisories].
